# WindowsEmergencyTool

Windows 应急响应辅助工具，开箱即用，无需安装任何运行库。

## 功能

- **用户排查** — 枚举系统用户账户，识别隐藏用户和克隆账户
- **进程排查** — 进程列表、进程树、DLL模块、签名校验、在线验证，支持过滤
- **网络连接排查** — TCP/UDP连接监控，外联IP提取，IP归属展示
- **启动项排查** — 注册表Run/RunOnce、启动文件夹、Winlogon劫持、IFEO等
- **服务排查** — 服务枚举与过滤，签名校验，快捷管理
- **计划任务排查** — 任务列举与过滤，关联文件提取，暂停/运行/删除
- **文件排查** — 恶意文件落地目录扫描，自定义路径与后缀
- **事件日志排查** — 登录、进程创建、日志清除等事件分析
- **威胁检索** — 基于IOC进程内存特征的威胁检索
- **活动痕迹** — UserAssist、Prefetch、最近访问等活动时间线

## 系统要求

- Windows 10 / 11（x64）
- 建议以管理员权限运行
- 无需安装运行库
